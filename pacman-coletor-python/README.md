🎮 Pacman Coletor - Jogo em Python

Este é um pequeno jogo desenvolvido em Python utilizando a biblioteca Pygame.

O jogador controla um personagem (Pacman) e precisa coletar pontos dentro de um limite de tempo.

🕹️ Como funciona

- Use as setas do teclado para mover o personagem
- Colete os pontos vermelhos
- O objetivo é alcançar **15 pontos antes que o tempo acabe**

⏱️ Regras do jogo

- Tempo limite: 30 segundos
- Pontos necessários para vencer: 15
- Cada ponto coletado aparece em uma posição aleatória

🧠 Conceitos praticados

Neste projeto foram utilizados:

- Python
- Biblioteca Pygame
- Lógica de jogos
- Movimentação de personagem
- Colisão entre objetos
- Sistema de pontuação
- Temporizador

▶️ Como executar o jogo

1. Instale o Python
2. Instale a biblioteca pygame
3. Execute o jogo
